양식태그

홈페이지를 사용하는 사용자가 내용을 입력할 수 있도록 양식을 제공할 때 사용하는 태그

label태그는 입력박스에 대한 이름표태그

label태그와 input태그와 같은 입력박스태그를 서로 연결하기 위해 input태그의 id속성과 label태그의 for속성 값을 똑같은 값으로 적용

input태그의 type속성에 적용되는 속성 값

  • text => 짧은 한 줄의 텍스트 입력
  • password => 비밀번호를 입력
  • radio => 필수적으로 선택해야할 항목(해제가 안됨)
  • checkbox => 자유롭게 선택할 항목(해제가 됨)
  • number => 숫자만 입력받는 용도로 사용(모바일환경 숫자패드가 올라옴)

사용되는 속성

  • value - 입력박스에 들어가는 실제정보/버튼형태에서는 버튼내용(자동로그인, 이전에 적었던 내용을 유지하기 위함)
  • disabled - 선택한 입력박스를 비활성
  • required - 선택한 입력박스 필수정보를 입력받도록(입력하지 않으면 전송시 안내문이 뜬다)
  • selected - option태그가 선택되도록
  • checked - radio/checkbox 버튼이 체크가 되도록
