가상클래스(추상클래스)

사용자의 행동양식을 선택자로 표현할 수 있다.

  • 선택자:hover - 사용자가 마우스를 올렸을 때의 스타일을 미리 지정할 수 있다
  • 선택자:active - 마우스를 클릭했을 때(클릭을 한 뒤에 마우스를 떼지 않은 상태)
  • 선택자:before - 선택한 태그의 첫번째 자식태그를 생성
  • 선택자:after - 선택한 태그의 마지막번째 자식태그를 생성

hover와 active는 주로 버튼에서 사용한다. 버튼이 작동하는 것에 대한 반응을 보여주기 위함

가상태그 before와 after태그는 css에서 일괄적으로 선택한 태그들에게 첫번째나 마지막 자식태그를 만들 때 사용하는 태그이다. display속성은 inline, p태그의 안에 만들어지는 태그, 드래그를 통해서 선택불가능, css에서만 수정가능

반드시 content라는 css속성이 명시되어 있어야 before태그와 after태그가 만들어진다.
