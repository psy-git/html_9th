선택자 점수

css는 선택자 부분의 표현이 특이점이 높을 수록 우선순위가 높게 적용된다.

우선순위가 높게 적용된다라는 것은 css스타일이 최종적으로 적용된다는 뜻이다.

단계적으로 적용되는 css

브라우저는 위에서 아래 왼쪽에서 오른쪽으로 소스코드를 해석한다 그렇기 때문에 나중에 적용한 스타일 속성이 최종 적용되는 속성이 된다.

선택자의 우선순위 점수

태그를 선택하는 선택자는 우선순위 점수가 높을수록 최종스타일로 적용된다(우선순위점수는 합산된다), 동일한 스타일에서 특정한 부분의 스타일만을 수정하기 위해서 클래스를 추가적으로 적용해준다.

  • 태그명으로 선택 - 1점 자릿수 증가
  • 클래스명으로 선택 - 10점 자릿수 증가
  • 아이디명으로 선택 - 100점 자릿수 증가
  • 인라인스타일방식 - 1000점 자릿수 증가
  • !important - 무조건 적용
